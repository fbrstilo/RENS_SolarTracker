RENS Solar Tracker use guide

ADMIN

In order to access this page You need to input the Admin password.
This page is used for initial communication setup and device default values

LOGIN

This page is used to elevate user privileges to ADMIN.
Change the password in the Admin control panel.
To reset the password follow the instructions in the RESET_ADMIN_PASS file (located on the web server).

DEVICE CONTROL

To access this page click on the Devices dropdown in the sidebar menu and choose the device you wish to control

Controls available to all users:

Angle/Panel Tilt control:

MAKE SURE TO SWITCH BACK TO AUTO WHEN MANUAL CONTROL IS NO LONGER NEEDED
Automatic control: set the toggle to the 'Auto' position and press the 'Send data' button
Manual control: set the toggle to the 'Manual' position, write the wanted angle in the text input area, and press the 'Send data' button.
The device will never move past the east/west limits

Request log:

Create a new .log file, viewable in the 'Logs' section
To request the device's last log, switch the toggle to the 'Last' position and press the 'Request log' button
To request all available logs, switch the toggle to the 'All' position and press the 'Request log' button. This action takes 8x longer than requesting a single log

Reset:

Reboot the device. After reboot the device will sound off the alarm and move to its home position, before resuming automatic control

Controls available to the Admin:

To revert values to their default, submit the data with empty input fields.

Siren and insolation:


Siren on time - system boot warning duration in seconds.
Insolation - max area of next solar panel that the controlled solar panel is allowed to 'hide' from the Sun. Expressed as a percentage:
 0% - next panel is fully hidden from the Sun
 100% - next panel is fully exposed to the Sun

Global position:

Set the latitude and longitude of the system. This data is used together with the current time in order to calculate the appropriate elevation angle.

Time offset:

Set the device time offset in seconds.
This data is used together with the current UTC time to make adjustments for the actual time zone.

Angle limits:

Set the maximum east/west panel angle. This is used to protect the system from physical damage. The system will never exceed these angles, even if manually ordered to

Height difference:

Set the heights of this (first) and next (second) panel, as viewed from the Sun. This data is used to calculate the insolation of the next panel

Panel length and axis distance:

Set the panel length and distance between panel axes. This data is used to calculate the insolation percentage of the next panel

Home position and motor revolutions per degree

Home position - the first position to which the device goes to after booting up
Motor RPD - ratio between motor revolutions and solar panel angle change

LOGS

All system logs will appear in this menu. Log viewing is available to all users.
